# LP-CODE

Personal website for my YouTube channel and courses.

## Features
- **YouTube Channel**: Links, videos, and content updates.
- **Courses**: Interactive listing and information about courses.
- **Modern UI/UX**: Premium responsive design built with React, Vite, and TailwindCSS.

## Getting Started

### Prerequisites
Make sure you have Node.js installed.

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
